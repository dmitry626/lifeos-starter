# Семья

Всё, что связано с семьёй, домом, детьми, имуществом, бытом.

## Что здесь хранить

**Дети:**
- Образование (школы, курсы, репетиторы, расписание)
- Воспитание (правила, договорённости, подходы)
- Активности (секции, кружки, лагеря)
- Здоровье (прививки, аллергии, врачи -- или ссылка на Здоровье/)

**Имущество:**
- Дом / квартира (документы, обслуживание, ремонт, IT-инфраструктура)
- Автомобиль (документы, ТО, страховка)
- Дача / загородный дом

**Документы:**
- Паспорта, визы, разрешения (сроки действия, когда продлевать)
- Страховки (полисы, покрытие, контакты)

**Персонал (если есть):**
- Няня, помощник, домработница -- контакты, договорённости, расписание

**История рода:**
- Генеалогия, семейный архив

## Что AI может помочь

- Напомнить о сроках продления документов
- Подготовить чек-лист для переезда или ремонта
- Систематизировать информацию по школам при выборе
- Отслеживать расписание детских активностей
