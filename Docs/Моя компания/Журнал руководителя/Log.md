# Журнал руководителя

Хронология ключевых решений и событий. Не дневник -- только то, что важно помнить через месяцы.

Формат записи:

```
### YYYY-MM-DD -- Заголовок

Что произошло / что решили / почему.
Ссылки на встречи, документы, людей.
```

---

### 2026-01-15 -- Пример записи

Приняли решение сфокусировать продажи на сегменте X. Причина: анализ показал 3x конверсию vs текущий сегмент. Ответственный: [[Имя Фамилия]].
