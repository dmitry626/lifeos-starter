# Здоровье

Медицинская информация, чек-апы, врачи, привычки здоровья.

## Что здесь хранить

**Ежегодный чек-ап:**
- Результаты анализов (по годам -- для отслеживания динамики)
- Показатели: холестерин, сахар, давление, вес, ИМТ
- Рекомендации врачей

**Врачи и клиники:**
- Контакты проверенных специалистов (терапевт, стоматолог, офтальмолог...)
- Где что лечили, кто помог

**Прививки:**
- Календарь прививок (особенно важно для детей)
- Что сделано, что предстоит, сроки ревакцинации

**Стоматология:**
- План лечения, визиты, снимки

**Питание:**
- Принципы питания, диеты, ограничения
- Аллергии и непереносимости

## Что AI может помочь

- Сравнить результаты анализов по годам, показать тренды
- Напомнить о плановых визитах
- Найти информацию о показателях (нормы, что значит отклонение)
- Подготовить сводку для нового врача
