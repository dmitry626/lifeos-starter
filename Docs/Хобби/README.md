# Хобби

Всё, что приносит удовольствие и не является обязанностью. Время для себя.

## Что здесь хранить

**Спорт:**
- Серфинг, гольф, теннис, фитнес, йога -- по папке на увлечение
- Тренировки, прогресс, контакты тренеров
- Споты, маршруты, сезоны

**Музыка:**
- Гитара / другой инструмент (аккорды, песни, уроки)
- Плейлисты, рекомендации

**Путешествия:**
- Маршруты, заметки, рекомендации по странам
- Wish-list мест

**Что угодно ещё:**
- Фотография, кулинария, настольные игры, коллекционирование
- Правило: если увлечение "выросло" в десятки файлов -- выноси в отдельную папку на уровне Docs/

## Что AI может помочь

- Составить план тренировок или маршрут путешествия
- Найти споты для серфинга / гольф-поля по критериям
- Подобрать аккорды или табулатуру для песни
- Систематизировать заметки с поездок
