# Финансы

Всё, что связано с деньгами: доходы, расходы, счета, налоги, инвестиции, стратегия.

## Что здесь хранить

**Бюджет и расходы:**
- Семейный бюджет (доходы/расходы по категориям, лимиты)
- Подписки и регулярные платежи
- Крупные покупки (план, сравнение)

**Банковские счета:**
- Какие счета, в каких банках, в каких валютах
- Реквизиты (без паролей -- только "где лежит доступ")

**Налоги:**
- Ежегодный чек-лист сбора отчётности
- Дедлайны по юрисдикциям
- Контакты бухгалтера / налогового консультанта

**Инвестиции:**
- Портфель (структура, доли)
- Стратегия (горизонт, принципы, ребалансировка)

**Финансовая стратегия:**
- Долгосрочные цели (пассивный доход, подушка безопасности)
- Принципы (сколько откладывать, куда, зачем)

## Что AI может помочь

- Проанализировать расходы по категориям
- Напомнить о дедлайнах по налогам
- Сравнить условия банков или инвестиционных продуктов
- Подготовить данные для бухгалтера
