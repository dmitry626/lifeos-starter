# Личное развитие

Цели, привычки, обучение, рефлексия, работа с коучем/психологом.

## Что здесь хранить

**Цели:**
- Годовые цели (что хочу достичь в этом году)
- Долгосрочные цели (3-5-10 лет)
- Ретроспектива целей (что удалось, что нет, почему)

**Привычки и трекеры:**
- Ежедневные практики (медитация, спорт, чтение)
- Марафон привычек / трекер

**Коучи и психологи:**
- Записи сессий, задания, инсайты
- Практики и упражнения

**Книги и обучение:**
- Что читаю / прочитал (ссылки на карточки книг в Resources/Библиография/)
- Ключевые выводы и как применяю
- Курсы, конференции

**Рефлексия:**
- Еженедельные ретроспективы
- Дневник (или ссылка на Daily Notes/)

## Что AI может помочь

- Провести еженедельную ретроспективу (собрать факты, задать вопросы)
- Подготовить саммари книги с ключевыми идеями
- Помочь сформулировать цели (SMART, OKR)
- Отследить прогресс по привычкам
