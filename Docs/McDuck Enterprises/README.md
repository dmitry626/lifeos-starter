# McDuck Enterprises

## О компании

- **Роль:** Основатель, генеральный директор
- **Год основания:** 1885
- **Команда:** ~20 человек (склады, приисков, администрация)
- **Штаб-квартира:** Дакбург

## Продукты и направления

| Направление | Описание | Статус |
|-------------|----------|--------|
| Золотодобыча | Прииски на Клондайке и в Южной Африке | Основной доход |
| Торговля | Снабжение приисков, торговые маршруты | Запускается (проект [[01-Projects/002 Торговый маршрут Дакбург — Клондайк/README|002]]) |
| Золотохранилище | Хранение и управление активами | Действует |

## Стратегия

1. Вертикальная интеграция: добыча → логистика → хранение → финансы
2. Экспансия на новые территории (Клондайк — первый шаг)
3. Подготовка наследников к управлению
