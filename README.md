# LifeOS — персональная операционная система

Единое место для проектов, знаний, встреч, решений и задач. Обычные текстовые файлы на твоём компьютере + AI-ассистент, который видит всё и помогает управлять.

## Быстрый старт

### 1. Установить Claude Code

```bash
# macOS / Linux
curl -fsSL https://claude.ai/install.sh | bash

# Windows (PowerShell)
irm https://claude.ai/install.ps1 | iex
```

Требуется подписка Claude Pro/Max: [claude.ai](https://claude.ai) -- Settings -- Billing.

### 2. Установить Obsidian

Скачать: [obsidian.md](https://obsidian.md) (бесплатно). Открыть эту папку как vault.

### 3. Запустить

```bash
cd ~/LifeOS    # или где лежит эта папка
claude
```

При первом запуске Claude Code проведёт интервью: спросит, кто ты, чем занимаешься, какие задачи. По результатам настроит систему под тебя.

### 4. Создать алиас (необязательно)

```bash
# Добавить в ~/.zshrc (macOS) или профиль PowerShell (Windows)
alias lifeos='cd ~/LifeOS && claude'
```

## Структура папок

```
CLAUDE.md           — инструкция AI-ассистенту (настраивается под тебя)
README.md           — этот файл
00-Goals/           — цели жизни (мечты, год, квартал)
00-Inbox/           — входящее (всё, что нужно разобрать)
01-Projects/        — активные проекты (каждый: папка + README)
99-Archive/         — завершённые проекты
Daily Notes/        — ежедневные записи (дневник)
Docs/               — области ответственности (бизнес, финансы, здоровье...)
Persons/            — люди (контакты, контекст)
Resources/          — справочные материалы (книги, журнал сессий)
Templates/          — шаблоны заметок
```

## Что делать после установки

Стартовый проект уже создан: `01-Projects/001 Онбординг LifeOS/`. Откройте его — там пошаговый план от первого запуска до ежедневной работы.

| День | Что сделать |
|------|------------|
| 1 | Пройти онбординг, описать свой бизнес, заполнить 2-3 карточки людей |
| 2-3 | Обработать первую встречу, создать свой проект, поставить цели |
| 3-5 | Подключить задачник и календарь через MCP |
| 7+ | Вести Daily Notes, разбирать Inbox, записывать итоги сессий |

## Команды

| Команда | Что делает |
|---------|-----------|
| `/meeting` | Обработать встречу: саммари, решения, задачи |
| `/inbox` | Разобрать входящие из `00-Inbox/` |
| `/session-log` | Записать итоги работы с ассистентом |

## Демо-данные

Система содержит демонстрационные данные от имени Скруджа Макдака:
- Проект "Торговый маршрут Дакбург — Клондайк" с встречами и артефактами
- Цели (от мечты до квартального плана)
- Карточки людей (племянники, партнёры, конкуренты)
- Daily Notes, входящие, книга, журнал руководителя

Это примеры — замени их своими данными при онбординге.

## Принципы

- **Файлы на твоём компьютере.** Не в облаке чужой компании.
- **AI работает с контекстом.** Чем больше описано — тем точнее ответы.
- **Система растёт вместе с тобой.** Начни с минимума, добавляй по мере необходимости.
- **Один файл — одно место.** Не дублировать, а ставить ссылку.
