# LifeOS — персональная операционная система

Единое место для проектов, знаний, встреч, решений и задач. Не приложение, а методология + набор инструментов, которые ты настраиваешь под себя.

## Как устроена система

**База знаний (Obsidian)** — здесь живёт всё важное: проекты, встречи, решения, идеи, контакты. Обычные текстовые файлы на твоём компьютере, связанные ссылками.

**AI-ассистент (Claude Code)** — видит всю базу знаний и работает с ней: создаёт проекты, обрабатывает встречи, анализирует информацию, подключается к почте и календарю. Не нужно копировать файлы в чат — он видит всё.

## Быстрый старт

### 1. Установить Claude Code

```bash
# macOS / Linux
curl -fsSL https://claude.ai/install.sh | bash

# Windows (PowerShell)
irm https://claude.ai/install.ps1 | iex
```

Требуется подписка Claude Pro ($20/мес): [claude.ai](https://claude.ai) -- Settings -- Billing.

### 2. Установить Obsidian

Скачать: [obsidian.md](https://obsidian.md) (бесплатно). Открыть эту папку как vault.

### 3. Запустить

```bash
cd ~/LifeOS    # или где лежит эта папка
claude
```

При первом запуске Claude Code проведёт интервью: спросит, кто ты, чем занимаешься, какие задачи. По результатам настроит систему под тебя.

### 4. Создать алиас (необязательно)

```bash
# Добавить в ~/.zshrc (macOS) или профиль PowerShell (Windows)
alias lifeos='cd ~/LifeOS && claude'
```

## Структура папок

```
CLAUDE.md           — инструкция AI-ассистенту (настраивается под тебя)
README.md           — этот файл
00-Inbox/           — входящее (всё, что нужно разобрать)
01-Projects/        — активные проекты (каждый: папка + README)
99-Archive/         — завершённые проекты
Docs/               — области ответственности (бизнес, финансы, здоровье...)
Resources/          — справочные материалы
Templates/          — шаблоны заметок
```

## Команды

| Команда | Что делает |
|---------|-----------|
| `/meeting` | Обработать встречу: саммари, решения, задачи |
| `/project new Название` | Создать новый проект |
| `/project work YYYY-NNN` | Загрузить контекст проекта для работы |
| `/session-log` | Записать итоги сессии |

## Частые ошибки

1. **Пытаться перенести всё сразу.** Начни с 2-3 проектов и Inbox.
2. **Создавать сложную структуру до того, как она нужна.** Папки появляются по потребности.
3. **Дублировать информацию.** Одна единица знания — в одном месте. Из других — ссылка.
4. **Забывать про CLAUDE.md.** Без него ассистент не знает контекст.

## Принципы

- **Файлы на твоём компьютере.** Не в облаке чужой компании. Синхронизация — через iCloud, OneDrive или Obsidian Sync.
- **AI работает с контекстом.** Чем больше ты описал — тем точнее ответы.
- **Система растёт вместе с тобой.** Начни с минимума, добавляй по мере необходимости.
