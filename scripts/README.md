# Скрипты

Скрипты автоматизации LifeOS.

## Трекинг сессий

| Скрипт | Назначение | Вызывается |
|--------|-----------|------------|
| `session-tracker.sh` | Хук: логирует timestamp+TZ при каждом промпте | `.claude/settings.json` (автоматически) |
| `session-analyze.sh` | Анализ лога: активное время, паузы, локация, параллельные сессии | `/session-log`, `/session-review` (вручную) |

### Зависимости

- `jq` — парсинг JSON в хуке (есть fallback на grep)
- `python3` — конвертация timestamp с учётом часового пояса
- macOS или Linux (логи в `/tmp/`)

### Как это работает

1. При каждом промпте `session-tracker.sh` дописывает строку `2026-03-21T08:14:59+0800` в `/tmp/claude-session-{session_id}.log`
2. При вызове `/session-log` Claude запускает `session-analyze.sh` для расчёта метрик
3. Метрики включаются в запись журнала `Resources/Session Log/`
4. `/session-review` читает понедельные файлы и показывает паттерны

### Настройка

**Порог паузы** (когда интервал между промптами считается перерывом):
```bash
# session-analyze.sh, строка 3:
PAUSE_THRESHOLD=900  # 15 минут в секундах
```

**Маппинг локаций** (какой часовой пояс = какой город):
```bash
# session-analyze.sh, функция tz_to_location():
+0500) echo "Мой город (UTC+5)" ;;
+0800) echo "Удалённый офис (UTC+8)" ;;
```

### Временные файлы

Логи сессий хранятся в `/tmp/claude-session-*.log`. Удаляются автоматически при перезагрузке. Между перезагрузками накапливаются, но весят меньше 1 КБ каждый.
