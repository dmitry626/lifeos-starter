Управление проектами. $ARGUMENTS

## Подкоманды

Разбери `$ARGUMENTS`:
- `new` — создание нового проекта
- `work` — загрузка контекста проекта для работы
- Пусто — покажи список активных проектов из `01-Projects/README.md`

---

## /project new

Создаёт новый проект.

1. Прочитай `01-Projects/README.md`, возьми `last_project_id`, прибавь 1.
2. Спроси название проекта (если не передано в аргументах).
3. Создай папку `01-Projects/NNN Название/` с README.md по шаблону `Templates/Project Template.md`.
4. Обнови реестр: добавь строку, обнови `last_project_id`.

---

## /project work

Загружает контекст проекта для работы.

1. Найди проект по ID или названию.
2. Прочитай README проекта.
3. Покажи коротко: статус, что в работе, список артефактов и встреч.

---

## Правила

- Русский язык, без эмодзи.
- Ссылки: `[[wikilink]]`, без `.md`.
- Реестр проектов: `01-Projects/README.md`.
- Шаблон: `Templates/Project Template.md`.
- Архив: `99-Archive/`.
