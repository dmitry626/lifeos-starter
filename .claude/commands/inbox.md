Разбери инбокс. $ARGUMENTS

## Назначение

Разбор папки `00-Inbox/` — рассортировать входящее по местам. Каждый элемент: решить, что с ним делать, и переместить или удалить.

## Алгоритм

1. Прочитай все файлы в `00-Inbox/`.
2. Для каждого элемента определи:
   - **Что это?** (идея, задача, документ, мусор, справочная информация)
   - **Куда?** (проект, область в Docs/, Persons/, Resources/, удалить)
   - **Нужно действие?** (да — создать задачу, нет — просто разместить)

3. Предложи маршрутизацию каждого элемента. Формат:

```
### [Название файла]
- Тип: идея / задача / документ / мусор
- Куда: [путь назначения]
- Действие: [создать задачу / переместить / удалить / оставить на потом]
- Комментарий: [почему именно туда]
```

4. **Не перемещать и не удалять без подтверждения.** Показать весь план, дождаться "да" или корректировок.

5. После подтверждения — выполнить: переместить файлы, создать задачи (если подключен задачник), обновить связи.

## Правила

- Если элемент непонятен — спросить пользователя, а не додумывать.
- "Мусор" — только очевидный (дубли, пустые файлы, устаревшее). Если сомневаешься — не мусор.
- Inbox должен стремиться к нулю, но не за счёт качества разбора.
- Русский язык, без эмодзи.
