## Цель (Goal)

Настроить систему LifeOS под себя и научиться ей пользоваться — от первого запуска до ежедневной работы с AI-ассистентом.

## Готово когда (Definition of Done)

- [ ] CLAUDE.md заполнен: имя, роль, компании, принципы
- [ ] Описана хотя бы одна компания в `Docs/`
- [ ] Создано 3+ карточки людей в `Persons/`
- [ ] Обработана хотя бы одна встреча через `/meeting`
- [ ] Написана первая Daily Note
- [ ] Поставлены цели на год и квартал в `00-Goals/`
- [ ] Подключена хотя бы одна интеграция (задачник или календарь)
- [ ] Проведён первый разбор входящих `/inbox`
- [ ] Записан первый `/session-log`

## План (Roadmap)

### Этап 1 — Знакомство с системой (день 1)

- [ ] Пройти онбординг: ответить на вопросы Claude Code, заполнить CLAUDE.md
- [ ] Изучить структуру папок: что где лежит, зачем нужно
- [ ] Описать свою компанию в `Docs/<Компания>/README.md`
- [ ] Заполнить 2-3 карточки людей (коллеги, партнёры) в `Persons/`
- [ ] Написать первую Daily Note

### Этап 2 — Первые процессы (дни 2-3)

- [ ] Обработать встречу: скинуть заметки или запись, получить саммари через `/meeting`
- [ ] Создать второй проект (настоящий, рабочий)
- [ ] Поставить цели: заполнить `00-Goals/3-Год` и `4-Квартал`
- [ ] Разобрать входящие: положить 3-5 элементов в `00-Inbox/`, затем `/inbox`

### Этап 3 — Интеграции (дни 3-5)

- [ ] Подключить задачник (Todoist, Bitrix24 или другой) через MCP
- [ ] Подключить Google Workspace (почта + календарь) через MCP
- [ ] Попробовать: "покажи мои письма за сегодня", "что в календаре на завтра"

### Этап 4 — Ежедневная работа (неделя 1-2)

- [ ] Вести Daily Notes ежедневно
- [ ] В конце рабочего дня — `/session-log`
- [ ] Обработать 2-3 встречи
- [ ] Через неделю — посмотреть журнал сессий и оценить, как идёт

## Ресурсы для изучения Claude Code

Официальные материалы Anthropic для понимания, как работает инструмент:

**Курсы (бесплатные):**
- [Anthropic Academy](https://www.anthropic.com/learn) — курсы по Claude, включая Claude Code
- [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action) — практический курс по работе с Claude Code

**Документация:**
- [Типовые сценарии работы](https://code.claude.com/docs/en/tutorials) — пошаговые инструкции: как исследовать файлы, обрабатывать данные, работать с изображениями
- [Возможности Claude Code](https://www.anthropic.com/claude-code) — обзор того, что умеет инструмент

**Что важно для не-программиста:**
- Claude Code работает с любыми текстовыми файлами, не только с кодом
- Команды пишутся обычным русским языком: "обработай эту встречу", "создай проект", "найди все письма от Иванова"
- Slash-команды (`/meeting`, `/inbox`) — это готовые сценарии, которые запускаются одним словом
- Символ `@` перед именем файла — быстрый способ показать файл ассистенту: `@README.md`
- `Shift+Tab` переключает режимы: обычный → автопринятие → режим планирования (только анализ, без изменений)
